# Astro Consensus Frontend

React and Vite client for Astro Consensus. See the repository-level
[`README.md`](../README.md) for setup, API, accuracy, and verification details.

```bash
npm ci
npm run dev
```

The development server proxies `/api` to `http://127.0.0.1:8000`. For a separate
deployed backend, set `VITE_API_URL` to its `/api` URL.
